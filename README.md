## üß† Design Principles Followed

Across all projects, the following principles are emphasized:

- **Single Responsibility Principle**
- **Open/Closed Principle**
- **Interface-driven design**
- **Loose coupling & high cohesion**
- **Readable and interview-friendly code**

The goal is to **explain design decisions clearly**, not to over-engineer.
